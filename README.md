# Learncpp.com Comments Anti-spam

A lightweight userscript that removes offensive or vertical "wall" spam from the LearnCpp.com comment sections (wpDiscuz).

## Features
- Detects and hides comments with excessive `<br>` tags or 1-letter-per-line spam
- Filters extreme slur or hate content
- Keeps normal discussion comments untouched
- Doesn't break any functionality.
